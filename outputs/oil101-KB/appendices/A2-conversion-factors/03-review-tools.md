# Appendix 2 - Conversion Factors: Review Tools

## Active Recall

1. 什么是 API from SG？
   - API = 141.5 / SG - 131.5。

2. 什么是 SG from API？
   - SG = 141.5 / (API + 131.5)。

3. 什么是 Barrels per tonne？
   - Approx = 1000 / (density kg/m3 * 0.158987)。

4. 什么是 HHV vs LHV？
   - HHV 包含水凝结热，LHV 不包含。

5. 什么是 BOE？
   - Barrel of oil equivalent，约 5.8-6.0 MMBtu。

6. 什么是 Mcf to BOE？
   - 常用 6 Mcf gas = 1 BOE。

7. 什么是 Precision rule？
   - 商业结算应使用合同规定 conversion。

# Chapter 7 - Refining: Review Tools

## Active Recall

1. 什么是 CDU？
   - 常压蒸馏按 boiling point 切分 crude。

2. 什么是 Vacuum distillation？
   - 在低压下分离重馏分，避免高温裂解。

3. 什么是 FCC？
   - 催化裂化把 gasoil 转成 gasoline-range molecules 和 LPG。

4. 什么是 Coker？
   - 热裂化重渣油，产 light products 和 petroleum coke。

5. 什么是 Hydrocracker？
   - 用 hydrogen 和催化剂裂化并脱硫，柴油质量好。

6. 什么是 Alkylation？
   - 把 isobutane 和 olefins 合成高 octane alkylate。

7. 什么是 Hydrotreating？
   - 用 hydrogen 去除 sulfur/nitrogen 等杂质。

8. 什么是 Nelson Complexity Index？
   - 衡量炼厂相对复杂度和升级能力。

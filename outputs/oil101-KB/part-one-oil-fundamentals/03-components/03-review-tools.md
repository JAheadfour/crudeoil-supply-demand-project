# Chapter 3 - Components of Oil Liquids: Review Tools

## Active Recall

1. 什么是 Total liquids？
   - 总供给口径包含多种液体而非只含 conventional crude。

2. 什么是 Refinery gain？
   - 炼厂输出液体体积大于输入 crude 体积，但质量守恒。

3. 什么是 Lease condensate？
   - 井口分离出的轻质液体，在统计上常归入 crude。

4. 什么是 NGL purity products？
   - Y-grade 经 fractionation 分成 ethane、propane、normal butane、iso-butane、natural gasoline。

5. 什么是 Ethane rejection？
   - 当 ethane 液体价值低于 gas BTU 价值时，处理厂会把它留在 gas stream。

6. 什么是 Dilbit？
   - Bitumen 与 diluent 混合后才能通过管线流动。

7. 什么是 Tight oil？
   - 页岩水平井和压裂生产的 light sweet oil。

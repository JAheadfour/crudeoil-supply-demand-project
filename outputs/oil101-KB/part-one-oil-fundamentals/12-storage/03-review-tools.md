# Chapter 12 - Storage: Review Tools

## Active Recall

1. 什么是 Contango storage value？
   - Forward price - spot price - storage cost - financing cost - losses。

2. 什么是 Backwardation？
   - 近月价格高于远月，持有库存的机会成本高。

3. 什么是 Days of cover？
   - Inventory / daily demand，用于衡量库存可支撑天数。

4. 什么是 Working capacity？
   - 可正常运营使用的库容，不等于名义总容量。

5. 什么是 SPR release？
   - 政府释放战略库存缓解短期供应冲击。

6. 什么是 Five-year range？
   - 用历史库存范围判断当前库存偏高或偏低。

7. 什么是 Floating storage？
   - 油轮被用作库存资产。

## Common Error Log

| Error | Correction |
|---|---|
| 只背术语，不看约束 | 每个概念都要放回质量、地点、时间、合同或监管背景中 |
| 把 financial price 当成 physical reality | 先确认是否有交割、库容、物流或规格限制 |
| 忽略区域差异 | Oil markets 是全球连接但局部受限的网络 |

# Chapter 8 - Standards: Review Tools

## Active Recall

1. 什么是 ASTM methods？
   - 标准化实验方法，使不同实验室结果可比。

2. 什么是 API gravity test？
   - 用密度推断原油轻重。

3. 什么是 RON？
   - Research Octane Number，较温和工况下的抗 knock 指标。

4. 什么是 MON？
   - Motor Octane Number，更高负荷工况下的抗 knock 指标。

5. 什么是 AKI？
   - Anti-Knock Index = (RON + MON) / 2，美国泵上标示。

6. 什么是 RVP？
   - Reid Vapor Pressure 衡量汽油挥发性。

7. 什么是 ULSD？
   - Ultra-low sulfur diesel，低硫让后处理催化器可用。

8. 什么是 IMO 2020？
   - 全球船燃 sulfur cap 改变 bunker fuel demand 和 refinery economics。

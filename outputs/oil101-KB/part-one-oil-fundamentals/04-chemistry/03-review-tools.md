# Chapter 4 - Chemistry of Oil: Review Tools

## Active Recall

1. 什么是 Methane combustion？
   - CH4 + 2 O2 -> CO2 + 2 H2O + heat。

2. 什么是 Iso-octane combustion？
   - C8H18 + 12.5 O2 -> 8 CO2 + 9 H2O + heat。

3. 什么是 Stoichiometric air-fuel ratio？
   - Iso-octane 接近 14.7:1 by weight。

4. 什么是 PONA？
   - Paraffins、Olefins、Naphthenes、Aromatics 是油品结构分类。

5. 什么是 Octane number？
   - 衡量汽油抗 knock 能力，branched paraffins 和 aromatics 通常高。

6. 什么是 Cetane number？
   - 衡量柴油压燃速度，straight-chain paraffins 通常高。

7. 什么是 Cracking reaction？
   - 长链分子可裂化为较短 paraffin 和 olefin。

8. 什么是 Alkylation？
   - Isobutane 与 olefin 结合生成高 octane alkylate。

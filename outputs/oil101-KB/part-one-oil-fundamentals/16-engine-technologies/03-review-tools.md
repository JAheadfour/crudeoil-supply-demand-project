# Chapter 16 - New Engine Technologies: Review Tools

## Active Recall

1. 什么是 Thermal efficiency？
   - Useful work / fuel energy input。

2. 什么是 Compression ratio？
   - Cylinder maximum volume / minimum volume。

3. 什么是 Octane requirement？
   - 发动机越激进，越需要高 octane fuel。

4. 什么是 Four-stroke cycle？
   - Intake、compression、power、exhaust 完成一次动力循环。

5. 什么是 Hybrid regeneration？
   - 回收制动能量并减少低效率工况。

6. 什么是 BEV efficiency？
   - 电驱动从电池到车轮效率高。

7. 什么是 Hydrogen fuel cell？
   - 氢与氧反应发电驱动电机。

8. 什么是 Fleet turnover？
   - 车辆存量更新慢，使燃料需求滞后于新车销售变化。

# Chapter 19 - Forward Oil Markets: Options: Review Tools

## Active Recall

1. 什么是 Call option？
   - 买方有权以 strike 买入标的。

2. 什么是 Put option？
   - 买方有权以 strike 卖出标的。

3. 什么是 Intrinsic value？
   - Option 若立即行权的价值。

4. 什么是 Time value？
   - Option price - intrinsic value。

5. 什么是 Delta？
   - Option value 对 futures price 的一阶敏感度。

6. 什么是 Gamma？
   - Delta 对价格变化的敏感度。

7. 什么是 Vega？
   - Option value 对 implied volatility 的敏感度。

8. 什么是 Collar？
   - 买 put 卖 call，降低 premium 并锁定区间。
